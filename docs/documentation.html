Getting Started

To use JawgStrap, you need to wrap your content in a block with id "jwg-strap".

<div id="jwg-strap">
HERE GOES YOUR CONTENT TO JAWGSTRAP !
</div>

Panel

Code 
<div class="container row space-between">
  <div class="panel">
    <h2 class="panel-title">Panel title</h2>
    This is a clickable Panel
  </div>
  <div class="panel flex-2">
    <h2 class="panel-title">Panel title</h2>
    This is a Panel with flex-2 weight
  </div>
  <div class="panel flex-3">
    <h2 class="panel-title">Panel title</h2>
    This is a Panel with flex-3 weight
  </div>
  <div class="panel noborder">
    <h2 class="panel-title">Panel title</h2>
    This is a Panel with no border
  </div>
</div>
CSS Specifications :
  • container
    • row : display panels in row direction
    • column : display panels in column direction
    • nowrap : avoid wrapping
    • centered : center the panels
    • space-between : panels take all place available in width with white space between
    • space-around : variant of space-between
    • panel
      • default : not flexible
      • flex-1 : weight of panel
      • flex-2 : weight of panel
      • flex-3 : weight of panel
      • flex-4 : weight of panel
      • flex-5 : weight of panel
      • panel-title
